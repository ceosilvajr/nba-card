### Initialize modules

- Run:`pip install openai`
- Run:`pip install python-dotenv`
- Run:`pip install reportlab`

### Open AI Key
- Add `.env` file
- Add your API-KEY and name it to `OPENAI_API_KEY`

### How to Run?

- Run: `python3 main.py`
